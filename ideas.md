# 株主優待カレンダー - デザイン案

## 案1: 親しみやすい北欧ミニマリズム
**デザイン運動**: 北欧ミニマリズム + ウォームモダン

**コアプリンシプル**
1. 親しみやすさ：丸みを帯びた要素と温かみのある色で初心者を迎える
2. 情報の明確性：ホワイトスペースを活用し、データを呼吸させる
3. 機能性：フラットデザインながら、微妙な影で階層感を表現

**色彩哲学**
- 背景：クリーム色（#FFFBF5）- 温かく、目に優しい
- プライマリ：やさしい青（#5B9BD5）- 信頼感と親しみやすさ
- 権利確定日：緑（#70AD47）- ポジティブな印象
- 権利落ち日：オレンジ（#FFA500）- 注意喚起だが温かい
- テキスト：濃いグレー（#333333）- 読みやすさ重視

**レイアウトパラダイム**
- ヘッダー：シンプルなロゴと月切り替えボタン
- メインエリア：カレンダーを左、フィルターパネルを右に配置
- カード型：企業情報を丸みを帯びたカードで表示

**シグネチャー要素**
1. 丸みを帯びた角（border-radius: 16px）
2. 優しい影（box-shadow: 0 2px 8px rgba(0,0,0,0.08)）
3. アイコン：Lucide Reactの親しみやすいアイコン

**インタラクション哲学**
- ホバー時：カードが軽く浮き上がる（shadow増加）
- クリック時：スムーズなページ遷移
- フィルター変更時：リスト更新がフェードイン

**アニメーション**
- ページ遷移：フェードイン（200ms）
- ホバー効果：shadow + scale（150ms）
- フィルター更新：リストのフェードイン（300ms）

**タイポグラフィシステム**
- 見出し：Noto Sans JP Bold（600-700）
- 本文：Noto Sans JP Regular（400）
- キャプション：Noto Sans JP Light（300）

## 案2: 洗練されたダークモダン
**デザイン運動**: ダークモダニズム + テック寄り

**コアプリンシプル**
1. 高級感：深いダーク背景で上質さを表現
2. 効率性：情報密度を高め、プロフェッショナルな印象
3. コントラスト：明るい要素で重要情報を強調

**色彩哲学**
- 背景：深紺（#0F1419）- 上質で落ち着き
- プライマリ：ネオンブルー（#00D9FF）- 現代的で目立つ
- 権利確定日：ネオングリーン（#00FF41）- 成功を示唆
- 権利落ち日：ネオンピンク（#FF006E）- 警告を示唆
- テキスト：ホワイト（#FFFFFF）- 高コントラスト

**レイアウトパラダイム**
- グリッドベース：2列レイアウト（カレンダー + フィルター）
- グラデーション背景：微妙なグラデーション効果
- ガラスモーフィズム：半透明のカード

**シグネチャー要素**
1. グラデーション枠線
2. グロー効果（box-shadow: 0 0 20px rgba(0,217,255,0.3)）
3. ネオンカラー

**インタラクション哲学**
- ホバー時：グロー効果が強調
- クリック時：ネオンカラーがフラッシュ
- フィルター変更時：スムーズなアニメーション

**アニメーション**
- ページ遷移：スライドイン（300ms）
- ホバー効果：グロー + scale（200ms）
- フィルター更新：スケールアップ（400ms）

**タイポグラフィシステム**
- 見出し：Inter Bold（700）
- 本文：Inter Regular（400）
- キャプション：Inter Light（300）

## 案3: 親しみやすい自然派デザイン
**デザイン運動**: オーガニックデザイン + アクセシビリティ重視

**コアプリンシプル**
1. 自然さ：有機的な曲線と自然色で親近感を醸成
2. アクセシビリティ：誰もが使いやすい設計
3. 温かさ：人間的で親しみやすい表現

**色彩哲学**
- 背景：オフホワイト（#F5F3F0）- 自然で柔らか
- プライマリ：アースグリーン（#6B8E6F）- 自然と信頼
- 権利確定日：フォレストグリーン（#2D5016）- 成長を示唆
- 権利落ち日：ウォームオレンジ（#D97706）- 注意を示唆
- テキスト：チャコール（#2C2C2C）- 読みやすさ重視

**レイアウトパラダイム**
- 非対称レイアウト：カレンダーを上部に大きく配置、下部に一覧
- 流動的：セクションが自然に流れる
- ホワイトスペース：呼吸感を重視

**シグネチャー要素**
1. 有機的な曲線（border-radius: 12-20px）
2. 自然なテクスチャ（微妙なノイズ）
3. 手書き風アイコン

**インタラクション哲学**
- ホバー時：背景色が自然に変化
- クリック時：スムーズで自然な遷移
- フィルター変更時：リストが優雅に更新

**アニメーション**
- ページ遷移：フェードイン（250ms）
- ホバー効果：背景色変化 + scale（150ms）
- フィルター更新：リストのスムーズな入れ替え（350ms）

**タイポグラフィシステム**
- 見出し：Noto Sans JP SemiBold（600）
- 本文：Noto Sans JP Regular（400）
- キャプション：Noto Sans JP Regular（400、色で区別）

## 選定理由
**案1（北欧ミニマリズム）を選択します。**

理由：
- 初心者向けの親しみやすさと、モダンなデザインのバランスが最適
- 温かみのある色合いで、投資という堅い話題を親しみやすく表現
- ホワイトスペースを活用した情報設計が、複雑なデータを整理するのに適している
- 日本ユーザーに好まれるNoto Sans JPとの相性が良い
- ダークモード対応も自然に実装できる
