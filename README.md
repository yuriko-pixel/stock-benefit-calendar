# 株主優待カレンダー

権利確定日と権利落ち日が一目でわかる、初心者向けの株主優待情報サイトです。

## 🎯 概要

株主優待の権利確定日・権利落ち日をカレンダーで視覚的に表示し、セクター・優待タイプ・投資額でフィルタリングして企業を探せるサイトです。初心者にも親しみやすいシンプルなデザインを採用しています。

## ✨ 主な機能

### 📅 カレンダー表示
- 当月の権利確定日と権利落ち日を色分けして表示
- 権利確定日：緑色
- 権利落ち日：オレンジ色
- 前後月への切り替え可能

### 🔍 フィルタリング機能
- **業種**：IT・情報通信、医療・介護、小売、製造業など10種類から選択
- **優待タイプ**：商品券、割引券、サービス、株主優待品、その他から選択
- **最低投資額**：10万円以下、10～50万円、50～100万円、100万円以上から選択
- 複数条件の組み合わせで絞り込み可能
- 選択条件をタグで表示し、個別削除または一括削除可能

### 📋 企業一覧表示
- 50社の株主優待情報を掲載
- 企業名、ティッカー、セクター、優待内容を表示
- 権利確定日・権利落ち日・最低投資額を明記

### 🔗 企業詳細ページ
- 企業をクリックして詳細情報を表示
- 企業の優待内容を詳しく確認可能
- SNS共有機能（Twitter、Facebook、LINE）

### 🌙 ダークモード対応
- ヘッダーのボタンでライト/ダークモードを切り替え可能
- 目の疲れを軽減し、夜間の閲覧に対応

## 🎨 デザイン

北欧ミニマリズムのデザイン哲学に基づいた、温かみのあるシンプルなUIです。

- **カラーパレット**：クリーム色背景（#FFFBF5）、やさしい青（#5B9BD5）、緑（#70AD47）、オレンジ（#FFA500）
- **フォント**：Noto Sans JP（親しみやすく読みやすい）
- **レイアウト**：カード型、丸みを帯びたコーナー、優しい影で視認性と親しみやすさを両立

## 🚀 セットアップ

### 必要な環境
- Node.js 22.13.0以上
- pnpm 10.4.1以上

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd stock-benefit-calendar

# 依存パッケージをインストール
pnpm install

# 開発サーバーを起動
pnpm dev
```

開発サーバーは `http://localhost:3000` で起動します。

### ビルド

```bash
# 本番用にビルド
pnpm build

# ビルド結果をプレビュー
pnpm preview
```

## 📊 データ構造

各企業の株主優待情報は以下の構造で管理されています：

```typescript
interface StockBenefit {
  id: string;                    // 企業ID
  name: string;                  // 企業名
  ticker: string;                // ティッカーシンボル
  sector: string;                // セクター（業種）
  benefitType: string;           // 優待タイプ
  description: string;           // 優待内容の説明
  exRightDate: string;           // 権利確定日（YYYY-MM-DD形式）
  exDividendDate: string;        // 権利落ち日（YYYY-MM-DD形式）
  minInvestment: number;         // 最低投資額（円）
  details: string;               // 詳細情報
}
```
